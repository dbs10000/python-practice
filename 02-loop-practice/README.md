Python Loop Practice
===============

OVERVIEW
--------
I wanted to practice Python loops.
The brief was to create a function that takes in a word or some text.
The output is a vowel count and the text in reverse order.

---

#### Why Did I Build This?
To practice getting my head around the logic of loops.

#### What I Have Learned:
* The range() tool is very handy when looping backwards.
* Using the input tool to allow a user to input a word or sentence of their choice.
* Practiced breaking my code into blocks and importing it back as modules.

----
#### HOW TO USE:
When you run the code it will ask you to input a word or some text. 
Once you hit enter, it generates the response
